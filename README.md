# Chukwuebuka Michael Umeh — Portfolio

A professional portfolio website showcasing projects, skills, certifications, and contact info. Built for internship and hiring visibility.

## Features

- **Hero** — Intro, typing animation, and “what I’m looking for” (internships, full-stack / AI)
- **About** — Bio, education, and highlights
- **Projects** — Featured work with live demos and GitHub links
- **Skills** — Core (shipped with) and Familiar tech
- **Certifications** — Oracle AI, Snap AR, CodePath, CompTIA, etc.
- **Contact** — Email (with copy button), LinkedIn, GitHub, Say Hello CTA
- **Resume** — Dedicated page to view or download resume PDF

## Tech Stack

- HTML5, CSS3, JavaScript (vanilla)
- [Font Awesome](https://fontawesome.com/) icons
- [Google Fonts](https://fonts.google.com/) — Inter, Fira Code
- No framework or build step

## Run Locally

```bash
# Install dependencies (optional — scripts use npx)
npm install

# Development (live reload, opens browser)
npm run dev

# Or static server on port 3000
npm start
```

Then open **http://localhost:3000** (or the port shown in the terminal).
